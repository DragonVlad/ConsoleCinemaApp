CONSOLE CINEMA APP ON JS

В этом приложении есть возможность хранения данных пользователя о просмотренных фильмах, актерах, жанрах. Также можно переключить доступность данных. Есть функции для определения
активности пользователя (по количеству фильмов), переключения доступности данных и определения любимых жанров пользователя.
