CONSOLE CINEMA APP ON JS

В этом учебном проекте я создаю небольшую БД, хранящуюся в объекте. В ней находятся количество просмотренных фильмов, сами фильмы с оценкой,
актеры этих фильмов, любимые жанры пользователя и триггер для переключения доступности БД. Используются методы для определения количества 
просмотренных фильмов, определения самих фильмов с оценкой, определения уровня зрителя (как много фильмов он посмотрел), определения 
любимых жанров пользователя и методы для переключения доступности БД.
